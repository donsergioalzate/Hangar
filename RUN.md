# Ejecutar el proyecto

1. Abre una terminal en la carpeta del proyecto:
   `C:\Users\Gabriel Revelo\OneDrive - Escuela Politécnica Nacional\Escritorio\Hangar`

2. Inicia el servidor de desarrollo (Windows):
   `cmd /c "set NODE_OPTIONS=--max-old-space-size=512 && npx next dev --hostname 0.0.0.0 --port 3000"`

3. Abre el navegador en:
   `http://localhost:3000`
